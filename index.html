[ITT MÁSOLD BE A LECKE TARTALMÁT]

    2. Hibaüzenetet kaptam

    A következő hibaüzenetet kaptam a terminálban (vagy a Claude Code-ban):

[ITT MÁSOLD BE A HIBÁT]

Mit jelent ez magyarul, miért történt, és pontosan mit csináljak, hogy megoldjam? Lépésről lépésre, a legegyszerűbb megoldástól kezdve.

    3. Készítek képernyőképet és azt mutatom

    Windows billentyűkombináció:
    Win + Shift + S → megjelenik a Képmetsző (Snipping Tool) → kijelölöd a területet → a kép a vágólapra kerül
    Vagy nyomj PrtScn billentyűt → a teljes képernyő a vágólapon

    Utána a Claude Desktop vagy Claude Code input mezőjébe: Ctrl + V beillesztés.
